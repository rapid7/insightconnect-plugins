# Description

The [Solarwinds](https://www.solarwinds.com/service-desk) application is a cloud-based IT service management platform. The ITIL-ready platform includes incident, problem, release and change management modules.

The Solarwinds servicedesk plugin allows you to start workflows on new incidents, manage incidents, and manage users.

This plugin utilizes the [Solarwinds Service desk API](https://www.samanage.com/api).

# Key Features

* Manage users
* Manage incidents
* Trigger new workflows on new incidents

# Requirements

* A Solarwinds Service Desk API token

# Supported Product Versions

* Solarwinds Service Desk with API version 2.1

# Documentation

## Setup

The connection configuration accepts the following parameters:

|Name|Type|Default|Required|Description|Enum|Example|
|----|----|-------|--------|-----------|----|-------|
|eu_customer|boolean|False|True|Is the customer based in Europe?|None|False|
|token|credential_secret_key|None|True|API Token generated by Solarwinds admin|None|1.2345563454676776e+27|
|ssl_verify|boolean|True|True|Boolean property used to decide whether to verify a TSL or SSL certificate|None|True|

Example input:

```
{
  "eu_customer": false,
  "token": 1.2345563454676776e+27,
  "ssl_verify": True
}
```
